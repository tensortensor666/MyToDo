# MyTodo 产品规划

## 📱 产品定位

**本地优先的 Todo 应用，支持多设备同步**

- 🏠 **本地优先**: 数据存储在本地，无需联网即可使用
- 🔄 **局域网同步**: 通过 LAN 实现设备间快速同步
- ☁️ **云端备份**: 可选 Supabase 远程同步
- 🖥️ **跨平台**: Windows、Android、iOS、Web

## 🎯 核心功能 (v1.3.4)

### ✅ 已实现
- [x] 任务管理（创建、编辑、完成、删除）
- [x] 任务分类（我的一天、重要、已计划、自定义清单）
- [x] 截止日期和提醒
- [x] 日常模板（每日自动生成任务）
- [x] 历史搜索（包含已删除任务）
- [x] 局域网同步（二维码配对）
- [x] Supabase 远程同步
- [x] Windows 托盘
- [x] 自动更新检查
- [x] Fluent UI（Windows 11 风格）

## 🎨 UI 改善计划

### 立即执行 (本周)

#### 1. 视觉层次优化
**问题**: 当前界面层次感不够明显

**解决方案**:
```dart
// 增加卡片阴影和边框
Container(
  decoration: BoxDecoration(
    color: theme.resources.cardBackgroundFillColorDefault,
    borderRadius: BorderRadius.circular(8),
    border: Border.all(
      color: theme.resources.cardStrokeColorDefault,
    ),
    boxShadow: [
      BoxShadow(
        color: Colors.black.withOpacity(0.04),
        blurRadius: 8,
        offset: Offset(0, 2),
      ),
    ],
  ),
)
```

#### 2. 间距优化
**问题**: 部分元素间距过小，视觉拥挤

**改进**:
- 任务卡片间距: 2px → 8px
- 页面边距: 8px → 16px
- 标题与内容间距: 8px → 16px

#### 3. 图标视觉效果
**问题**: 图标颜色不够突出

**改进**:
- 为不同类型任务使用不同颜色图标
- 增加图标悬停效果
- 提升重要标记的视觉权重

#### 4. 空状态优化
**当前**: 简单的文字提示

**改进**:
- 添加插画或图标
- 引导性文案
- 快速操作按钮
